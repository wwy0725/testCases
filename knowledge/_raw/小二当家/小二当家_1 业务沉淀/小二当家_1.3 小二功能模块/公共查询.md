### [查询功能](#查询功能)

-查询全国范围的企业，显示交互同[查企业pc](/read/book/92/192464)

### [客户信息展示](#客户信息展示)

- 选中客户后，客户信息卡片展示以下字段：
    - 客户名称
    - 用户代码
    - 行政区划
    - 责任归属
    - 是否会员
    - 产品活跃
    - 关联多企业
- 责任归属格式为【中心】大区 网点【客户经理姓名】
- 无关联多企业时显示"-"
- 无活跃记录时产品活跃字段显示"-"
- 产品活跃信息展示：最新活跃渠道、最新活跃功能、最新活跃时间
- 非当前入口调用接口不展示活跃信息
- 会员标签数据展示：会员、逾期会员、非会员

### [重置功能](#重置功能)

- 点击重置按钮清空输入框内容
- 客户类型恢复默认为"单位客户"
- 客户信息模块也清空

### [查询范围](#查询范围)

- 支持查询全国范围

## [接口信息](#接口信息)

### [搜索客户接口](#搜索客户接口)

**接口地址：** /customer/searchCustomerByContent

**请求方式：** POST

**请求参数：**

```plaintext
{
    "provinceCityAreaCode": "all",  // 查询全国范围
    "queryCount": 20,             // 返回结果数量限制
    "keyword": "搜索关键词",        // 企业名称/信用代码/税号
    // 其他参数同首页搜索
}

```

### [查询企业信息接口](#查询企业信息接口)

**接口地址：** /customer/getCustomerInfo

```plaintext
{
    "customerId": "客户ID",
    "queryLatestActiveFlag": "Y"  // 是否查询最新活跃信息，默认不查询
}

```
