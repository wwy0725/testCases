#### 新增企业

不可添加场景：

| - | - | - |
|---|---|---|
| 类型 | 原因 | 交互 |
| 已添加 | 企业已经在当前经营体ID关联的企业列表中（不考虑是否和当前登陆用户所属同一分子公司代码等其他场景，只要在关系中就算「已添加」） | / |
| 已关联正式多企业 | 企业已在其他多企业中 | 按企业添加时点击“已关联正式多企业”跳转至对应多企业一户式  按人添加时不可添加企业列表内点击“已关联正式多企业”跳转至对应多企业一户式  导入添加时展示“已关联正式多企业：多企业名称” |
| 有经营中心且非登录人经营中心且不存在多企业和企业的有效订单 | 只可以添加与登录人同经营中心或无经营中心企业或与登录人不同经营中心但存在多企业的有效订单的企业  有效订单：企业关联有该多企业的委托单，但订单状态为正常，收款状态为无需收款/部分收款/已收讫 | 返回该企业的经营中心  若不可添加，原因为"企业无关联订单，无法跨中心添加" |
| 已注销 | - | 返回已注销 |

不可添加原因优先级：

企业状态为已注销。
企业已添加到当前多企业中。
企业已关联到其他正式多企业。
企业无关联订单，无法跨中心添加（定义：企业有经营中心属性，且该中心 ≠ 当前操作用户的经营中心，且不存在对应多企业的有效关联订单）。

##### 按企业添加

查询全集团所有企业，展示企业名称、税号 / 社会信用代码（即一户式用户代码）、归属经营中心（若有）、注销状态（若有）、不可选原因（若有）

搜索企业判断是否符合添加条件，不符合的企业展示失败原因且置灰，符合的企业可点击，点击后默认选中。点击“已关联正式多企业”跳转至对应多企业一户式

搜索企业时符合提醒条件的企业展示提醒文案（提醒条件：该企业可添加到多企业，但不属于当前操作用户的责任田且最近30天内存在跟进记录）

##### 按人添加

搜联系人同首页顶部的按人查询，查询全集团范围的关联个代，展示联系人姓名、手机号、个代标记、个代名称、地区、会员类型、责任田归属

联系人非个代时，获取全部自主企业列表

联系人是个代时，获取自主企业+个代省直单下代理企业列表

自主企业和代理企业的逻辑同一人式

获取企业列表后批量校验是否符合添加条件，不可添加的可点击打开弹框展示具体的失败原因，点击“已关联正式多企业”跳转至对应多企业一户式

选中人后展示人的关联企业中可添加的企业列表时，符合提醒条件的企业展示提醒文案（提醒条件：该企业可添加到多企业，但不属于当前操作用户的责任田且最近30天内存在跟进记录）

##### 导入添加

校验文件行数是否超过最大行数（IRIS配置：import.file.max.row.count，目前线上配置10000）
校验文件大小上限（IRIS配置：import.file.max.size，目前线上配置1M）

根据导入文件的客户代码查询全集团的企业，校验名称是否相符，不相符则提示“客户不存在”。根据客户代码查询不到客户则调用客户中心接口[新增单位客户dubbo接口](/read/book/70/170664)自动创建客户，创建失败提示“新客户自动创建失败，无法关联”。校验失败的企业可下载失败文件，企业已关联正式多企业时展示多企业名称。

导入创建客户时设置增值纳税人类型为其他，导入文件中的税号/社会信用代码传入到税号字段，customerCreationScene为multiEnterpriseAutoCreate，其他字段以导入文件为准。
