# 销售机会相关表

| - | - | - | - |
|---|---|---|---|
| itcrm\_system\_config | 添加销售机会产品包选择模式配置表 | key：editProductPackageType，"value":"components"（产品包模式）， "value":"regular"（手动输入模式） | - |
| itcrm\_follow\_business\_last\_follow\_info | 销售机会最后跟进数据 | - | - |
| itcrm\_chance\_stage\_settings | 配置销售机会阶段 | 支持按大区配置，若大区未配置销售阶段则采用默认大区配置  配置阶段和百分比的对应关系。修改表里的数据后，需要调用refreshCache刷缓存。缓存每30分钟自动刷新 | - |
| itcrm\_chance\_package\_info\_syn | 销售机会关联的产品包数据 | 用于ES拉平产品包数据的表 | - |
| itcrm\_chance\_order\_relation | 销售机会订单关系表 | 保存赢单的销售机会关联的订单  销售阶段选为赢单时，必须要选择关联订单 | - |
| itcrm\_chance\_items | 销售机会跟进情况表 | 记录销售机会变更流水，销售机会每次操作变更都会记录，包括销售机会创建，编辑，跟进，客户释放/认领导致的销售机会责任人变更 | - |
| itcrm\_chance\_info | 销售机会基本信息表 | （1）保存销售机会关联的企业，创建人及所属部门，负责人及所属部门  （2）保存销售机会当前阶段值，若 stage\_code = win / fail 则表示销售机会为赢单 / 输单，此时 end\_status = Y，其他阶段code时 end\_status = N | - |
| itcrm\_chance\_collaboration\_info | 商机协作人员信息表 | - | - |
| itcrm\_chance\_item\_collaboration\_info | 商机协作人员信息动态表 | - | - |
