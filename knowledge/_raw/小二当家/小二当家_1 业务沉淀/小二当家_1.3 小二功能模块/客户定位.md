
### 1-数据存储

| - | - |
|---|---|
| 表 | 表说明 |
| itcrm\_location\_info | 定位信息表，保存定位详情信息及状态 |
| itcrm\_location\_info\_log | 定位日志表，记录定位变更操作日志 |

### 2-定位展示

根据**客户id+客户类型**[**04-获取定位信息**](/read/book/70/174443)

待确认的客户定位标红展示待确认提示文案（数据初始化产生的客户定位--通过客户的经营地址自动解析生成）

已确认的客户定位正常展示（编辑过的客户定位都为已确认）

确认定位 和 编辑定位的操作都对应[03-更新定位信息——已下线](/read/book/70/174444)

线索详情及客户详情点击去这里，跳转企微原生地图页面，显示定位，进行导航可正确唤起手机安装地图




### 3-编辑定位

[03-更新定位信息——已下线](/read/book/70/174444)同时更新定位日志

编辑定位后 CONFIRM\_STATUS = Y;AVAILABLE=Y

### 4-标记无效-财税

功能说明：客户定位状态有效时（AVAILABLE=Y），页面展示标记无效按钮，更新定位地址为无效（仅财税）[08-机构一户式定位标记无效\_已下线](/read/book/70/174440)

新增客户定位、编辑客户定位、自动解析生成的客户定位**默认定位状态都为有效**

| - | - |
|---|---|
| 定位状态 | 页面按钮展示 |
| 地址待确认；有效 | 确认定位；标记无效 |
| 地址待确认；无效 | 编辑定位 |
| 地址已确认；有效 | 编辑定位；标记无效 |
| 地址已确认；无效 | 编辑定位 |

### 5-同步客户经营地址-财税

功能说明：新增权限控制，有权限的客户经理操作编辑客户定位可以选择同步更新到客户中心的经营地址（仅财税）

功能权限code：**sysnUpdateBusinessAddress**

![image2022-8-29_15-18-34.png](https://snack.dc.servyou-it.com/snack/257/5ea708ac00d54a2fb573766276245ee2.png)

选择更新，改为当前定位地址，前端调用[03-更新定位信息——已下线](/read/book/70/174444)并传参数 checkUpdateAddress = Y，后端调用客户中心更新客户的dubbo接口，将定位地址保存到客户经营地址

### 6-解析客户mq信息

功能说明：配置需要自动解析客户经营地址的分公司，通过订阅客户中心的创建客户和编辑客户mq，解析经营地址为定位地址

配置项：itcrm\_system\_config 表

| - | - | - | - |
|---|---|---|---|
| 配置项 | 配置说明 | 消费的MQTopic | 解析MQ消息体 |
| SYNC\_CUSTOMER\_LOCATION | 单位客户--自动解析的分子公司配置 | cc-operation-customer-company-create  cc-operation-customer-company-update | 取mq消息体的bizRegionCode+businessAddress  （更新mq取 postChangeCompanyInfo 对象内的） |
| SYNC\_INSTITUTION\_LOCATION | 代理客户--自动解析的分子公司配置 | cc-operation-customer-agent-create  cc-operation-customer-group-update | 取mq消息体的bizRegionCode+address  （更新mq取 postChangeAgentInfo 对象内的） |

需要解析的经营地址调用腾讯地图解析，可解析成经纬度的保存至 itcrm\_location\_info 表，并在 itcrm\_location\_info\_log 表记录变更日志
