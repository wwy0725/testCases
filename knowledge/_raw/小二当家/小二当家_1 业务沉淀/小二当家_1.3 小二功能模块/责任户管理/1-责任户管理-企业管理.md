### **创建待办**

- 有权限“批量创建待办”权限的，我的企业/下属的企业/网点的企业的列表展示待办创建入口对接待办组件，跨页全选上限5000超出报错。
- 跨页全选场景下，前端传入当前列表查询条件调用/todo/myCustomerDimensionBatchCreateTodo，接口内部查询当前筛选条件下全部客户数据后传给待办接口生成待办任务id，前端获取到待办返回的id传入待办组件创建待办
- 跨页全选反选场景，前端传入当前筛选条件+未选择的客户id（excludedCustomerIdList）
- 单选多选单页全选等其他场景前端直接传入客户id调用/todo/myCustomerDimensionBatchCreateTodo获取到待办任务id后对接待办组件
- 创建待办组件前端对接入参如下

```java
{
    "options": {
        "rwlyid": "43665b82c889484c8555995642012c26",
        "source": "itcrmweb-mycustomer",//来源场景默认我的企业，若外部打开企业列表指定待办来源场景则传入指定的场景如经营看板
        "appCode": "itcrmweb",
        "todoProperty": "company",
        "env": "test",
        "showHead": false
    }
}
```
